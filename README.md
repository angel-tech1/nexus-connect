# Nexus Connect

Kotlin back-end code for the Nexus Connect application.

## Tech Stack

- Kotlin back-end application codebase
- Gradle build configured with the Kotlin DSL
- Java toolchain targeting Java 21
- UTF-8 source encoding
- Terraform infrastructure files for AWS EC2 deployment modules

## Available Gradle Commands

This project does not define custom Gradle tasks yet. The standard Java/Gradle lifecycle tasks available at this time include:

- `gradle tasks`
- `gradle help`
- `gradle build`
- `gradle clean`
- `gradle test`
- `gradle check`
- `gradle assemble`
- `gradle jar`
- `gradle classes`
- `gradle compileJava`
- `gradle processResources`
